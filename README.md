# Zeus Cabinets & Countertops — Website

Source and version control for the [Zeus Cabinets & Countertops](https://zeuscabinetsflorida.com) website.

## About

Zeus Cabinets & Countertops designs, builds, and installs custom cabinetry and
countertops for residential spaces in Orlando and the greater Central Florida
area. The company specializes in custom, functional, and visually realistic
cabinetry design tailored to each home.

## Services

- Kitchen cabinets
- Bathroom vanities
- Laundry room cabinetry
- Closets
- Home office cabinetry
- Custom cabinets and storage solutions
- Countertops
- Professional design, delivery, assembly, and installation

## Service Area

Orlando, Florida, and Central Florida.

## Repository Status

This repository is being set up to maintain and version-control the existing
Zeus Cabinets & Countertops website. The site currently runs on **WordPress**.

This is **not** a rebuild — the existing WordPress files and database export
must be imported into this repository first. Until that import is complete:

- No theme, plugin, or hosting configuration has been added yet.
- The active theme, plugins, hosting setup, and PHP version are still to be
  identified once the current site is imported.

## Workflow

All development should happen against a local or staging copy of the site —
never directly against the production environment. Once the WordPress export
is imported, this section will be updated with setup and deployment
instructions.
